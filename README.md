## AI Systems

Buddy runs on **5 interconnected AI systems** that work together:

### 1. The Brain - Causal Intelligence Engine

The Brain doesn't just see patterns - it **understands causality**. It answers the question: *"WHY did this token pump?"*

- **Causal Reasoning** - Analyzes multiple factors to determine root causes
- **Evidence Collection** - Builds supporting evidence for each analysis
- **Confidence Scoring** - Calculates statistical confidence in its conclusions

```
Example Analysis:
├── Primary Cause: "Elite Wallet Signal" (40% weight)
├── Entry Timing: "Early" - 3.2s (HIGH contribution)
├── Market State: "Hot" (FAVORABLE)
└── Confidence: 87%
```

### 2. Wallet Intelligence - Deep Behavioral Profiling

Buddy builds a **psychological profile** for every wallet it encounters:

- **Trading Style Detection**: Sniper, Fast, Patient, FOMO
- **Risk Profile**: Aggressive, Moderate, Conservative
- **Win Rate & Multipliers**: Historical performance tracking
- **Narrative Preferences**: What types of tokens they prefer
- **Predictive Modeling**: Probability they'll buy a specific token

```
Wallet Profile: 87rR...AxJ
├── Win Rate: 73%
├── Style: Sniper (avg entry: 4.2s)
├── Avg Multiplier: 8.3x
├── Favorite: Meme tokens (67%)
└── Prediction: 85% chance to buy this token
```

### 3. Predator Engine - Hierarchical Decision System

The Predator Engine uses a **priority-based decision hierarchy**:

| Priority | Signal | Action |
|----------|--------|--------|
| 0 | Elite Wallet Detected | **AUTO-BUY** (bypass filters) |
| 1 | Coordinated Whale Buying | **AUTO-BUY** (3+ whales simultaneously) |
| 2 | Consecutive Whales | **AUTO-BUY** (2+ in a row) |
| 3 | All Filters Passed + AI 80%+ | BUY |
| 4 | Filters Passed (no AI) | BUY with caution |
| 5+ | Any Filter Failed | SKIP |

### 4. Adaptive Agent - Self-Learning System

The Adaptive Agent **improves itself** over time:

- **Pattern Learning**: Tracks which signals lead to runners
- **Combo Discovery**: Learns powerful signal combinations
- **Market Mode Detection**: Hot, Cold, Normal, Volatile
- **Auto-Adjusting Thresholds**: Modifies its own rules based on results

```
Learning Example:
├── Pattern "Elite + Velocity" → Runner (12x)
├── Score Boost: 45 → 67 points
├── Combo "Triple Threat" activated
└── Success rate: 78% → 81%
```

**Exit Strategy Learning**:
- Tracks stop-loss exits that would have been winners
- Calculates optimal stop-loss based on real data
- Learns the average "dip before pump" pattern
- Adjusts recommendations automatically

### 5. Turbo Trader - Multi-RPC Racing

Speed is everything in trading. Buddy uses **parallel execution**:

- **Multi-RPC Racing**: Sends transaction to ALL RPCs simultaneously
- **First Wins**: Takes the first confirmation, cancels the rest
- **Jito MEV Integration**: Priority access to block builders
- **Dynamic Scoring**: Ranks RPCs by latency and success rate
- **Auto-Failover**: Instantly switches if an RPC fails

```
Transaction Race:
├── Helius: 127ms ✓ WINNER
├── QuickNode: 145ms (cancelled)
├── Triton: 162ms (cancelled)
└── Jito: 89ms (MEV bundle)
```
